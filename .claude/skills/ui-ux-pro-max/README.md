# ui-ux-pro-max skill (reference copy)

`SKILL.md` here is the **UI/UX Pro Max** ruleset (staged from the user's provided file). It's the accessibility / touch / performance / layout **rulebook and pre-delivery checklist** for the Yatri Cloud redesign — see the tables and "Pre-Delivery Checklist" sections.

> **Note:** the upstream skill also ships a `scripts/search.py` + `data/*.csv` search engine (50+ styles, 161 palettes, etc.). Those files were **not** provided, so the CLI search commands in `SKILL.md` (`python3 scripts/ui-ux-pro-max/scripts/search.py ...`) won't run here. The written guidance and checklists are fully usable as-is. To enable search, install the full skill from its source (`nextlevelbuilder/ui-ux-pro-max-skill`) into this folder.

Use it as the QA gate: before delivering any UI, run the checklist in `SKILL.md` (Accessibility, Interaction, Light/Dark, Layout).
