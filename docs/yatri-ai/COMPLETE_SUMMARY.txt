## 🚀 How to Get Started (5 Minutes)

### Step 1: Prerequisites
```bash
# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Download Gemma3 model
ollama run gemma3
```

### Step 2: Start Services (4 Terminals)
```bash
# Terminal 1: Ollama
ollama serve

# Terminal 2: Backend
cd certification.yatricloud.com
node server.js

# Terminal 3: Frontend
cd certification.yatricloud.com
npm run dev

# Terminal 4 (Optional): Verify
cd certification.yatricloud.com
bash verify-yatri-ai.sh
```

### Step 3: Use It
1. Open http://localhost:5173
2. Click the 💬 button in bottom-right corner
3. Type a message and press Enter
4. Get AI responses from Gemma3

## 📊 Architecture

```
Browser (http://localhost:5173)
    ↓
React App + YatriAI Component
    ↓ HTTP POST /api/chat
Express Server (http://localhost:3001)
    ↓ HTTP POST /api/generate
Ollama Service (http://localhost:11434)
    ↓ LLM Inference
Gemma3 Model (3B Parameters)
```

## 🔧 API Details

### Endpoint: POST /api/chat

**Request:**
```json

  "message": "What is AWS?"
}
```

**Response (Success):**
```json
{
  "response": "AWS (Amazon Web Services) is..."
}
```

**Response (Error):**
```json
{
  "error": "Ollama service unavailable",
  "details": "Make sure Ollama is running: ollama serve"
}
```
