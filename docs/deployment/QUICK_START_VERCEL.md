- ✅ Frontend updated to use Vercel API routes automatically

## 🎯 Next Steps (5 Minutes)

### 1. Go to Vercel (1 min)
- Visit: https://vercel.com
- Sign up/Login with GitHub

### 2. Import Project (1 min)
- Click **"Add New..."** → **"Project"**
- Select: `yatricloud/certification.yatricloud.com`
- Click **"Import"**

### 3. Add Environment Variables (2 min)
Click **"Environment Variables"** and add:

```
UDEMY_INSTRUCTOR_TOKEN=your_yatharth_token
UDEMY_INSTRUCTOR_TOKEN_NENSI=your_nensi_token
```

**Where to get tokens:**
- Go to: https://www.udemy.com/instructor/account/api/
- Generate tokens and paste here

### 4. Deploy (1 min)
- Click **"Deploy"**
- Wait 2-3 minutes
- Done! 🎉

## 🌐 Your Site Will Be Live At:
`https://your-project-name.vercel.app`
