## Column Headers

| Column # | Header | Data Type | Required | Description |
|----------|--------|-----------|----------|-------------|
| A | `timestamp` | Date | ✅ | Auto-generated timestamp when review is submitted |
| B | `name` | String | ✅ | Reviewer's full name |
| C | `feedback` | String | ✅ | Review text/feedback |
| D | `rating` | Number | ✅ | Star rating (1-5) |
| E | `linkedinProfile` | URL | ✅ | LinkedIn profile URL (https://linkedin.com/in/...) |
| F | `provider` | String | ✅ | Certificate provider (aws, azure, gcp, kubernetes, terraform, docker, github, salesforce, oracle, servicenow) |
| G | `source` | String | ✅ | Source of review (default: "web") |

---

## Data Types & Validation

### timestamp (Column A)
- **Type:** Date/Timestamp
- **Format:** Auto-generated by `new Date()` in Apps Script
- **Example:** `1/29/2026 6:45:30 PM`

### name (Column B)
- **Type:** String
- **Validation:** Required, non-empty
- **Example:** `John Doe`

### feedback (Column C)
- **Type:** String (Long Text)
- **Validation:** Required, non-empty
- **Example:** `This platform helped me understand cloud certifications. Highly recommended!`

### rating (Column D)
- **Type:** Number
- **Validation:** Required, integer between 1 and 5
- **Example:** `5`

### linkedinProfile (Column E)
- **Type:** URL
- **Validation:** Required, must match pattern `https?://(www.)?linkedin.com/.*`
- **Example:** `https://www.linkedin.com/in/johndoe/`

### provider (Column F)
- **Type:** String (Enum)
- **Validation:** Required, one of:
  - `aws` → AWS
  - `azure` → Azure
  - `gcp` → Google Cloud
  - `kubernetes` → Kubernetes
  - `terraform` → Terraform
  - `docker` → Docker
  - `github` → GitHub
  - `salesforce` → Salesforce
  - `oracle` → Oracle
  - `servicenow` → ServiceNow
- **Example:** `aws`

### source (Column G)
- **Type:** String
- **Validation:** Optional, defaults to "web"
- **Example:** `web`

## Apps Script Functions

### `addReview(data)`
Adds a new review row to the sheet.

**Input Parameters:**
```javascript
{
  name: "John Doe",              // Required
  feedback: "...",               // Required
  rating: 5,                     // Required (1-5)
  linkedinProfile: "https://...", // Required
  provider: "aws",               // Required
  source: "web"                  // Optional, defaults to "web"
}
```

**Returns:**
```javascript
{
  success: true,
  message: "Review added successfully"
}
```

### `getReviews(options)`
Retrieves all reviews with optional filters.

**Optional Parameters:**
```javascript
{
  minRating: 3,        // Filter: rating >= 3
  maxRating: 5,        // Filter: rating <= 5
  limit: 10            // Limit results to last N reviews
}
```

**Returns:**
```javascript
[
  {
    id: 2,                                          // Row number
    timestamp: "1/29/2026 6:45:30 PM",
    name: "Jane Smith",
    feedback: "...",
    rating: 5,
    linkedinProfile: "https://linkedin.com/in/janesmith/",
    provider: "aws",
    source: "web"
  },
  // ... more reviews
]
```

### `getReviewStats()`
Get overall statistics.

**Returns:**
```javascript
{
  totalCount: 42,
  avgRating: "4.8"
}
```

## HTTP Endpoints (Apps Script Web App)

**Base URL:** Set in `.env` as `VITE_CERTIFICATE_REVIEWS_APPS_SCRIPT_URL`

### POST - Add Review
```
POST /exec
Content-Type: application/json

{
  "name": "John Doe",
  "feedback": "Great course!",
  "rating": 5,
  "linkedinProfile": "https://linkedin.com/in/johndoe/",
  "provider": "aws"
}
```

**Response:**
```json
{
  "success": true,
  "message": "Review added successfully"
}
```

### GET - Retrieve Reviews
```
GET /exec?action=all&limit=10
```

**Response:**
```json
{
  "success": true,
  "data": [
    {
      "id": 2,
      "timestamp": "1/29/2026 6:45:30 PM",
      "name": "Jane Smith",
      "feedback": "...",
      "rating": 5,
      "linkedinProfile": "https://linkedin.com/in/janesmith/",
      "provider": "aws",
      "source": "web"
    }
  ]
}
```

### GET - Retrieve Stats
```
GET /exec?action=stats
```

**Response:**
```json
{
  "success": true,
  "data": {
    "totalCount": 42,
    "avgRating": "4.8"
  }
}
```

## Formatting Tips

1. **Header Row (Row 1):**
   - Bold text
   - Center alignment
   - Light gray background (#f0f0f0)

2. **Data Rows:**
   - Timestamp: Format as `m/d/yyyy h:mm:ss AM/PM`
   - Rating: Center alignment
   - Feedback: Text wrapping enabled

3. **Freeze Panes:** Freeze Row 1 for easy scrolling
