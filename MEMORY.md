- **D35 ‚Äî RFC 10008 HTTP QUERY METHOD (2026-07-03, commits 66fa358 + e369aa9).** RFC 10008 ("The HTTP QUERY Method", Proposed Standard June 2026 ‚Äî post-cutoff, VERIFIED via WebFetch before implementing) = safe/idempotent/cacheable GET-with-body.
