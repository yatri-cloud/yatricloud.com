# reference/ — visual inputs for the redesign

Drop supporting visual material here so Claude can look at it during design work.

| Folder | What goes here |
|---|---|
| `screenshots/` | Current Yatri Cloud screens (before-state) — capture key pages to redesign |
| `competitors/` | Screens from cloud-cert / EdTech / SaaS products worth learning from |
| `moodboards/` | Visual inspiration + craft references |
| `flows/` | User journeys and screen sequences (enroll, register for event, buy voucher) |
| `research/` | Usability notes, feedback, findings |

## Already staged
- `moodboards/DESIGN-lovable.md` — the **craft reference**. We borrow its discipline (opacity-driven neutrals, tactile depth, border-first containment, editorial spacing, tight display type), **not** its cream palette or Camera Plain font. Our palette stays the existing Yatri Cloud blue + monochrome.
